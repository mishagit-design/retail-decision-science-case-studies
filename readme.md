# Retail Decision Science Case Studies

## Overview

This repository contains a collection of retail analytics and decision science case studies inspired by real-world projects developed in a large-scale retail environment.

The projects focus on assortment optimization, customer analytics, recommendation systems, and cloud-based data modernization. While proprietary code and business data are excluded, each case study outlines the business problem, analytical approach, technologies used, and measurable outcomes.

## Case Studies

### Tiering Platform Modernization

Modernized a legacy assortment tiering platform by migrating complex business logic from Microsoft Access to Google BigQuery and Python-based workflows.

### Size Optimization System

Designed and maintained inventory allocation and size profile optimization processes supporting merchandise planning decisions.

### Customer & Store Profiling Framework

Developed behavioral segmentation approaches to support personalization and customer analytics initiatives.

### Sister Store Recommendation Engine

Built a store recommendation framework leveraging geographic and operational similarities to improve assortment decision-making.
